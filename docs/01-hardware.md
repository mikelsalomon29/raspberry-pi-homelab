\# Hardware



\## Raspberry Pi



\- Model: Raspberry Pi 3 Model B+

\- RAM: 1 GB

\- Storage: 16 GB microSD card

\- Network: Built-in Wi-Fi

\- Power: Micro-USB power supply



\## Setup Computer



\- Windows laptop

\- PowerShell used for SSH access

\- Raspberry Pi Imager used to flash Raspberry Pi OS



\## Current Hardware Limitations



The Raspberry Pi 3 Model B+ is suitable for lightweight server tasks, but it has limited RAM and storage. This homelab will focus on low-resource services such as Pi-hole, Tailscale, basic monitoring, and lightweight Docker containers.



Heavy services such as Jellyfin transcoding, Nextcloud, Kubernetes, and large databases are not planned for this hardware.
